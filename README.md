# Patient Management System

A full-stack patient management system built with HTML, CSS, JavaScript, Node.js, and Express.

## Features
- Add new patients
- View patient records
- Delete patient records
- Store data locally in a JSON file

## Project Structure
- `client/` for frontend
- `server/` for backend API and local data storage

## How to Run
1. Open terminal in the `server` folder
2. Run `node server.js`
3. Open `client/index.html` in your browser
